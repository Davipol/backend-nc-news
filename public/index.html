NEWS PROJECT API DOCUMENTATION

A Guide To Interact With News Project

Introduction

Welcome to News Project API documentation. This API allows the user to access different articles, comments, slugs and to vote articles.

Note: This API is publicly accessible and does not require authentication.

  • Base URL

    All API requests should be prefixed with the following base URL: 

    https://news-project-p4c3.onrender.com

Endpoints Overview

Here's a quick reference to all available API endpoints:

  • `GET /api` - Serves a JSON representation of all available endpoints.
  • `GET /api/topics` - Retrieves all topics.
  • `GET /api/articles` - Retrieves all articles, with optional filtering and sorting.
  • `GET /api/users` - Retrieves all users.
  • `GET /api/articles/:article_id` - Retrieves a single article by ID, including its comment count.
  • `GET /api/articles/:article_id/comments` - Retrieves all comments for a specific article.
  • `POST /api/articles/:article_id/comments` - Adds a new comment to an article.
  • `PATCH /api/articles/:article_id` - Updates the vote count for an article.
  • `DELETE /api/comments/:comment_id` - Deletes a specific comment.

Detailed documentation for each endpoint follows below.

Articles Endpoints

  • Get All Articles

    GET /api/articles

    Description:

    Serves an array of all articles.

    Optional Query Parameters:

    • sort_by: Sorts articles by a specific field.
      Valid options: created_at, votes, comment_count
      Default: created_at
    • order: Specifies the sorting direction.
      Valid options: asc, desc
      Default: desc
    • topic: Filters articles by topic slug.
      Example: coding, football

    These parameters can be appended as query strings, for example: 

    /api/articles?sort_by=votes&order=asc&topic=coding

    Example Response:

    {
      "articles": [
        {
          "title": "Seafood substitutions are increasing",
          "topic": "cooking",
          "author": "weegembump",
          "created_at": "2018-05-30T15:59:13.341Z",
          "votes": 0,
          "comment_count": 6
        }
      ]
    }

  • Get Articles By ID

    GET /api/articles/:article_id

    Description:

    Serves a single article with an added comment_count property containing the total comments

    Example Request:

    /api/articles/1

    Example Response:

    {
      "article": {
        "author": "butter_bridge",
        "title": "Living in the shadow of a great man",
        "article_id": 1,
        "body": "I find this existence challenging",
        "topic": "mitch",
        "created_at": "2020-07-09T20:11:00.000Z",
        "votes": 100,
        "article_img_url": "https://images.pexels.com/photos/158651/news-newsletter-newspaper-information-158651.jpeg?w=700&h=700",
        "comment_count": 10
      }
    }

  • Add Votes To An Article

    PATCH /api/articles/:article_id

    Description:

    Updates the vote count of an article by the specified amount.

    Example Request:

    Example Response:

      }

Comments Endpoints

  • Get Comments By Article ID

    GET /api/articles/:article_id/comments

    Description:

    Serves all comments for a given article based on the article_id

    Example Request:

    /api/articles/1/comments

    Example Response:

    {
      "comments": [
        {
          "comment_id": 5,
          "votes": 0,
          "created_at": "2020-11-03T21:00:00.000Z",
          "author": "icellusedkars",
          "body": "I hate streaming noses",
          "article_id": 1
        }
      ]
    }

  • Insert A Comment To An Article

    POST /api/articles/:article_id/comments

    Description:

    Adds a comment to the specified article. Requires a request body and with username and body fields.

    Example Request:

    {
      "username": "butter_bridge",
      "body": "This morning, I showered for nine minutes."
    }

    Example Response:

    {
      "comment": {
        "comment_id": 45,
        "body": "This morning, I showered for nine minutes.",
        "article_id": 2,
        "author": "butter_bridge",
        "votes": 0,
        "created_at": "2020-11-03T21:00:00.000Z"
      }
    }

  • Delete A Comment

    DELETE /api/comments/:comment_id

    Description:

    Deletes the comment with the specified comment_id.

    Example Request:

    /api/comments/2

    Example Response:

    {}
